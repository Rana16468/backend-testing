### Installation and running the app

Clone the repository:

```sh
git clone https://github.com/your-username/your-project.git
```

Navigate to the project directory:

```sh
cd project_name
```

Install dependencies:

```sh
yarn install | npm install
```

Star the app:

```sh
yarn start
```
